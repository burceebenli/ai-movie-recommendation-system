# AI Movie Recommendation System

This project is a movie recommendation system built using the TMDb API and machine learning techniques.

## Features

- Movie search
- AI based recommendation system
- Movie posters and descriptions
- Web interface with Streamlit

## Technologies

- Python
- TMDb API
- Streamlit
- Scikit-learn
- REST API

## How to run

pip install -r requirements.txt
streamlit run app.py
